# cs-portfolio
CS 210 Portfolio Projects
This repository contains two C++ projects completed in CS 210: a Calculator program and a Clock program.

These projects demonstrate my foundational understanding of C++ programming, including control flow, modular design, formatted output, and problem-solving through structured logic.
The Calculator program was designed to perform mathematical operations based on user input.

It required careful implementation of conditionals, function separation, and logical processing to ensure calculations were accurate and efficient. The Clock program maintains and displays time in both 12-hour and 24-hour formats while allowing users to increment hours, minutes, and seconds. 
A key challenge in this project was implementing rollover logic so that seconds correctly advance into minutes and hours without breaking the time structure. Working through this helped strengthen my understanding of state management and conditional sequencing.

One area I performed well in was organizing both programs into clearly separated functions, which improved readability and maintainability. I focused on consistent formatting, descriptive variable names, and inline comments to align with professional coding standards. Testing increment logic step-by-step in the Clock program also helped ensure reliable behavior.

There are several improvements I would make in future iterations. I could enhance input validation to prevent invalid entries, implement stronger exception handling, and further develop object-oriented structure in the Calculator program. These changes would make the programs more secure, scalable, and aligned with industry best practices.

Through these projects, I strengthened skills that are transferable to future coursework and software development work, including debugging strategies, modular program design, structured logic development, and writing maintainable C++ code. These assignments reinforced the importance of clarity, organization, and incremental testing when building functional software.
